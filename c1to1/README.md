

# Chat-1to1/1对1聊天


* Installation

```
    cnpm install
```

* dependencies

```
    "mysql": "^2.16.0",
    "socket.io": "^2.0.3"
```

* client files (for test)

```
    /c1demo.*
```

* Operation/操作
  - import `_data.sql` to mysql db
  - config `cfgs.js`
  - cmd (run under windows)
  - cd c1to1 
  - node server.js
  - view `demo.htm` on browser and test...
